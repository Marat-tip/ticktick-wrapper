new Task(properties)

Task model

Parameters
properties:Object

Properties of the Task

properties.id:string = ObjectID() (optional)

Object ID of the tasks, only defined when this is an instantiation of a pre-existing tasks. On new taskss this must be empty in order to generate a new ObjectID

properties.title:string

Title of the task

properties.content:string (optional)

Description of the task

properties.startDate=:Date|string (optional)

When the task is set to start. If it is a string it must have the format YYYY-MM-DDTHH:mm:ss.sss+0000 or YYYY-MM-DDTHH:mm:ss.sssZ

properties.dueDate=:Date|string (optional)

When the task is set to end. If it is a string it must have the format YYYY-MM-DDTHH:mm:ss.sss+0000 or YYYY-MM-DDTHH:mm:ss.sssZ

properties.timeZone:string (optional)

Timezone used for the triggering of the task reminders and assigned date. If empty it will be set to the account's default.

properties.isAllDay:boolean = False (optional)

Whether the task is set to take the entire day

properties.priority:Task.Priority = Task.Priority.NONE (optional)

Priority of the task

properties.status:Task.Status = Task.Status.TODO (optional)

Status of the task

properties.items:Array.<Object> = [] (optional)

Items of the task

properties.reminder:Reminder (optional)

Closest reminder

properties.reminders:Array.<Reminder> = [] (optional)

Reminders of the task

properties.progress:Number (optional)

Progress of items, from 0 to 100

properties.kind:Task.Kind = Task.Kind.TEXT (optional)

Task kind

properties.creator:string (optional)

User id of the task creator

properties.listId:string (optional)

List in which the task belongs

properties.repeatFlag:string (optional)

A set of flags that make a task repeatable
